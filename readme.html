In this repository are present some information about the modelling of CRM entities by Villa I Tatti in the context of the Art History Data programme.

The choices are expressed in Modelling.md and, based on that, rendered as a webpage available at the address https://ncarboni.github.io/arthistorydata/

Built with Couscous using a theme based on Read the Docs.
